= Overview

This is a basic implementation of fb303 for Ruby.

= Usage

See ./examples/server

= What You Get

Implementations for:

* getName
* getVersion
* getStatus
* getStatusDetails
* aliveSince
* getCounters
* getCounter
* shutdown
* Automatic per-method counters for:
  * success
  * exceptions (total)
  * exceptions (per exception)
